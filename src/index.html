Your Static Website

This is a template website which is built using Parcel. From here, you can do various things:

  • Delete everything and start from scratch, starting with a 
    index.html
    file,
  • Add on to the existing boilerplate code (there isn't a lot)
  • Add on other goodies such as 
    scss
    , 
    typescript
     (included, just create a 
    .ts
    file), and even some 
    react
    !

Batteries-included commands (run them with Yarn or NPM):

  • start
    : starts the dev server 
  • build
    : builds a production bundle ready for deployment. Artifacts are located in 
    ./dist/
